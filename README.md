# FreeAPI YouTube Videos Listing UI - Neo Brutalism

Web Dev Cohort 2026 Project

![Screenshot](screenshot.png) <!-- Add screenshot manually -->

## Features
- Neo Brutalist design: Chunky borders, hard shadows, red accents, raw hover effects
- Fetches real videos from [FreeAPI YouTube Videos](https://api.freeapi.app/api/v1/public/youtube/videos)
- Responsive grid (1-4 cols)
- Click thumbnail to open YouTube
- Loading/error states, formatted views/duration
- Pure HTML/CSS/JS + Tailwind CDN

## Quick Start
1. Open `index.html` in browser or:
   ```
   npx live-server . --port=8080
   ```
   View: http://localhost:8080

## API Response Handling
JS adapts to common fields (title, channel, thumbnail.url, views, duration, id). Adjust `script.js` if needed.

## GitHub Repo & Live Demo

   git init

   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/youtube-videos-ui.git
